# 🎲 Tenzies Game (React)

A simple implementation of the Tenzies dice game built with React.
This project was created to practice React fundamentals such as state management, components, and event handling.

## 🚀 Features

- Roll 10 dice
- Hold dice to keep their value
- Roll until all dice show the same number
- Win detection
- Clean and responsive UI

## 🛠 Technologies Used
- React
- Vite
- JavaScript
- CSS

## 🎮 How to Play

1. Click Roll to roll all dice.

2. Click a die to hold its value.

3. Keep rolling until all dice show the same number.

4. When all dice are the same and held, you win!
